## Featured Projects

A collection of my technical projects spanning AI/ML, embedded systems, and data engineering. Each project represents a unique challenge and learning opportunity in my journey as an engineer.

### 🤖 AI & Machine Learning

**Self-Healing Multi-Agent System for EDA**
Research thesis project at KU Leuven in collaboration with GSK. Built an intelligent system for automated exploratory data analysis in the pharmaceutical domain using multi-agent architecture and Generative AI.

**LLM Discovery Multi-Agent System**
Advanced multi-agent system implementation for automated discovery processes. Leverages Large Language Models to create intelligent, scalable information retrieval workflows.

**IEEE Papers Analysis & Summarization**
NLP system for analyzing and summarizing academic papers on power electronics. Automated classification of research trends in Inverters, DC-DC Converters, and On-board chargers with data visualization.

**Master Thesis: NLP & Word2Vec**
Text Statistical Analysis and correlation with Google's Word2Vec Dictionary using bare-metal C programming. University of Patras Master's thesis exploring fundamental NLP concepts at the lowest level.

### 🔧 Engineering & Hardware

**Router Integration Testing Framework**
Pip-installable integration testing framework for network setup automation. Streamlined functional tests on routers with comprehensive validation toolkit using Python and Pytest.

**Battery Management System PCB**
Designed PCB from scratch for 4-motor formula racing car's battery management system. First of its kind in FSAE Greece history. Includes 3D enclosure design, cost engineering, and competition-ready documentation.
